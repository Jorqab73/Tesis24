Instrucciones para descargar shape file para mapa en R.

Entra a pa siguiente dirección en tu navegador: https://www.ngdc.noaa.gov/mgg/shorelines/data/gshhg/latest/

En PracUni7/sperm_whale/data,  descarga el archivo gshhg-bin-2.3.7.zip y descomprímelo. 

El shapefile con el que trabajaremos se encuentra en el directorio GSHHG/i y se llama GSHHS_i_L1.shp
